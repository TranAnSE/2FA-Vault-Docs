# 2FA-Vault Docs

Repository used to edit the public documentation for 2FA-Vault.

2FA-Vault is a personal fork of 2FA-Vault. It keeps the upstream self-hosted OTP vault behavior and documents fork-specific work around end-to-end encryption, encrypted backups, teams, web push, and browser-extension encrypted vault sync.

2FA-Vault main repository: <https://github.com/TranAnSE/2FA-Vault>

## Validation

Install dependencies, then build the Retype documentation into `.retype-build`:

```bash
npm install
npm run validate:docs
```
